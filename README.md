## üöÄ How to Run a Project

Since these are separate projects, you need to run them individually.

1.  Clone the repository:
    ```bash
    git clone [https://github.com/](https://github.com/taco0cat/React-Basics.git)
    ```
2.  Navigate to the specific project folder:
    ```bash
    cd "01 - Basic-React-App"
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```
4.  Start the server:
    ```bash
    npm run dev
    ```
